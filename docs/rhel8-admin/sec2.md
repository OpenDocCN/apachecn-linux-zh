# 第二部分：SSH、SELinux、防火墙和系统权限的安全性

生产系统中的安全是系统管理员的直接责任。 为了处理这个问题，RHEL 包括 SELinux 等功能，这是一个集成的防火墙，当然还有标准的系统权限。 在本节中，我们将对 RHEL 中的安全机制进行全面的概述和理解，以便您能够执行日常维护任务。
